# Night City

A shell script to show night cities.

To run

```
git clone https://github.com/arjunair86/NightCity.git
cd NightCity
chmod +x city.sh
./city.sh <no_of_buildings> <maximum_height>
```

Example:

```
./city.sh 50 10
```
